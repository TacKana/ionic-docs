使用 Capacitor 为您的应用添加原生功能，这是由 Ionic 团队构建的原生运行时。安装核心包并轻松将它们添加到您的项目中。Capacitor 拥有广泛的功能，供开发者使用以访问诸如设备文件系统、摄像头和原生定位服务等特性。所有这些都由一个统一的 TypeScript API 提供支持，它能自动处理平台差异。

虽然 Capacitor 的核心功能是免费和开源的，但一些企业可能会发现自己需要更多功能或定制的第三方集成。如果您需要此类附加功能，请查看 [Ionic Enterprise SDK](https://ionic.io/enterprise-sdk)
