[WebNative Visual Studio Code 扩展](https://marketplace.visualstudio.com/items?itemName=WebNative.webnative) 是一个社区维护的插件，它可以帮助您执行常见的 Ionic Framework 开发任务，而无需记住 CLI 命令。

如果您的计算机上安装有 VS Code，请点击下方安装。您也可以通过搜索 "WebNative" 来找到此扩展。

</a>

</a>

## 更多文档

您可以在 [webnative.dev](https://webnative.dev/introduction/getting-started/) 找到 WebNative 扩展的完整文档，涵盖以下主题：

- 构建、调试和运行您的 Ionic Framework 应用程序。
- 包分析、依赖项升级。
- 从 Cordova 迁移。
- 更改原生设置。
- 闪屏和图标。
- 使用 WebNative 应用程序在没有 Mac 的情况下进行开发。
