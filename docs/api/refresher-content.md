刷新内容组件包含在下拉刷新（pull-to-refresh）过程中显示的文本、图标和旋转加载器。Ionic 会根据当前平台显示相应的下拉图标和刷新加载器。不过，您可以根据刷新器的状态自定义默认的图标、加载器和文本。

有关使用示例，请参阅 [Refresher（刷新器）](/api/refresher) 文档。
