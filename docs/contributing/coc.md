# 行为准则

作为 Ionic 项目的贡献者和维护者，我们承诺尊重每一位通过发布问题、更新文档、提交拉取请求、在评论中提供反馈以及参与任何其他活动做出贡献的人。

通过 Ionic 的任何渠道（GitHub、Slack、论坛、IRC、邮件列表、Twitter 等）进行的沟通必须是建设性的，绝不诉诸人身攻击、恶意挑衅、公开或私下骚扰、侮辱或其他不专业的行为。

我们承诺对这一项目中涉及的每个人保持礼貌和尊重，无论其性别、性别认同、性取向、残障状况、年龄、种族、民族、宗教信仰或经验水平如何。我们期望所有为 Ionic 项目做出贡献的人都能做到同样的事情。

如果社区中有任何成员违反了本行为准则，Ionic 项目的维护者可能会采取相应措施，包括删除问题、评论和 PR，或在认为适当时封禁账户。

如果您遭受或目睹了不可接受的行为，或有任何其他疑虑，请通过 <a href="mailto:devrel@ionic.io">devrel@ionic.io</a> 联系我们。

请点击 <a href="https://ionic.io/code-of-conduct" target="_blank" rel="noopener">此处</a> 查看 Ionic 的完整行为准则。
