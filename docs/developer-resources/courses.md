### [使用 Ionic 和 Electron 构建桌面应用](https://pluralsight.pxf.io/VeMXO)

<!-- cspell:disable-next-line -->

作者：Michael Callaghan（于 Pluralsight）

历史上，桌面开发所需的技能与 Web 开发所需的技能大不相同。这两门学科融合得并不好。在本课程《使用 Ionic 和 Electron 构建桌面应用》中，你将能够将辛苦学到的 Web 开发技能应用到构建出色的桌面应用中。首先，你将学习如何使用 Ionic 和 Angular 构建功能完善且美观的用户界面。接下来，你将探索如何将该用户界面封装到 Electron 应用程序外壳中。最后，你将学习如何打包你的应用并使其准备好分发。完成本课程后，你将具备 Ionic 和 Electron 开发所需的技能和知识，能够向 Windows 和 macOS 用户部署和分发美观的应用程序。

### [使用 Ionic 构建渐进式 Web 应用](https://pluralsight.pxf.io/Ly2EY)

<!-- cspell:disable-next-line -->

作者：Michael Callaghan（于 Pluralsight）

当谷歌提出渐进式 Web 应用程序（PWA）的概念时，一切都改变了。PWA 是一种纯粹的 Web 应用程序，你可以将其安装在设备上，通过智能缓存的使用，即使在网络功能有限的情况下也能运行。构建一个可以在任何地方运行的渐进式 Web 应用。在本课程《使用 Ionic 构建渐进式 Web 应用》中，你将学习基础知识，并获得创建能够在浏览器、桌面或移动客户端上运行的 Web 应用程序的能力。首先，你将了解什么是渐进式 Web 应用（PWA）。接下来，你将探索如何使用 Ionic 框架、Angular 和 Firebase 将一个基本的 Web 应用程序创建、部署和优化为一个成熟的 PWA。最后，你将学习如何配置应用程序，使其能够在 Android 和 iPhone 上安装和运行。完成本课程后，你将具备 Ionic 和 PWA 所需的技能和知识，能够在任何你想要的地方创建和部署你自己的渐进式 Web 应用程序。

### [Ionic CLI](https://pluralsight.pxf.io/ionic-cli)

<!-- cspell:disable-next-line -->

作者：Michael Callaghan（于 Pluralsight）

自诞生以来，Ionic 框架就包含了一个基本的命令行界面。虽然只有几年的历史，但它已经发展成为一个强大的工具，应该成为每个开发者工具箱的一部分。本课程《Ionic CLI》将从顶层开始探索 Ionic CLI。首先，你将看到如何创建项目和组件。接下来，你将学习如何构建和运行应用。最后，你将发现如何与他人共享项目，甚至如何与其他构建工具集成。无论你是刚刚开始探索 Ionic，还是从它的预发布阶段就开始使用，这里都有适合你的内容。课程结束时，你将充满信心地将 Ionic CLI 作为日常 Ionic 开发的一部分。
