# Ionic Angular 概览

`@ionic/angular` 将 Ionic 的核心体验与专为 Angular 开发者定制的工具链和 API 相结合。

## Angular 版本支持

Ionic 支持 `Angular 6.0.0 及更高版本`。作为其升级策略的一部分，Angular 内置了工具链来帮助自动化升级，并在 API 发生变化时向开发者提供反馈。这减少了更新阻力，并使生态系统保持在常青状态。

## Angular 工具链

从 Ionic 4+ 开始，我们使用官方的 Angular 技术栈来构建应用和处理路由，因此您的应用可以与优秀的 Angular 生态系统保持一致。在需要更多定制化功能的情况下，Ionic 提供了 `@ionic/angular-toolkit`，该工具包基于[官方 Angular CLI](https://angular.io/cli) 构建并与之集成，为 `@ionic/angular` 应用提供特定功能。
