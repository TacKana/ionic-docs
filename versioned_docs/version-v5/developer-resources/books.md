# 书籍推荐

### [从零开始学习 Ionic 4](https://leanpub.com/learnionic4fromscratch)

Angular、Vue、React、原生 JavaScript。借助全新的 Stencil 编译器，所有这些工具都可以与 Ionic 结合，打造出卓越的应用程序。本书面向初学者，旨在通过多种流行框架的示例，指导读者使用 Ionic 创建出色的 Web、移动和桌面应用。

<!-- cspell:disable-next-line -->

作者：[Paul Halliday](https://developer.school)

### [使用 StencilJS 构建 Ionic 应用](https://www.joshmorony.com/creating-ionic-applications-with-stencil-js/) - [免费试读](https://cdn2.hubspot.net/hubfs/3776657/PREVIEW-Creating-Ionic-Apps-with-StencilJS.pdf)

StencilJS 不仅是生成可复用 Web 组件的强大工具，还提供了构建完整 Web 组件应用所需的一切。结合 Ionic Web 组件，StencilJS 为我们构建高质量的生产级移动应用提供了完整解决方案——无需任何框架。

<!-- cspell:disable-next-line -->

作者：[Joshua Morony](https://www.joshmorony.com/blog)

### [Ionic 移动应用开发：使用 Ionic 2、Angular 2 和 Cordova 构建跨平台应用](https://www.amazon.com/Mobile-App-Development-Ionic-Cross-Platform/dp/1491937785/ref=sr_1_2?ie=UTF8&qid=1464183332&sr=8-2&keywords=ionic+2)

作者：[Joshua Morony](https://www.joshmorony.com/blog)

《使用 Ionic 和 Angular 构建移动应用》是学习 Ionic 最新版本的权威指南。本书面向初学者，从 Ionic 基础知识入手，逐步讲解不同复杂度的示例应用，最后详细说明如何构建并发布应用（到应用商店或作为 PWA）。本书会随每个主要版本更新，确保读者学到的都是最新技术。
