# ionic capacitor update

更新 Capacitor 原生平台，安装 Capacitor/Cordova 插件

```shell
ionic capacitor update [options]
```

`ionic capacitor update` 将执行以下操作：

- 更新 Capacitor 原生平台及依赖项
- 安装所有发现的 Capacitor 或 Cordova 插件
