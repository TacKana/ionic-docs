# ionic capacitor add

为你的 Ionic 项目添加原生平台支持

```shell
ionic capacitor add [options]
```

`ionic capacitor add` 命令将执行以下操作：

- 在你的项目中添加一个新的平台特定文件夹（ios、android 或 electron）
