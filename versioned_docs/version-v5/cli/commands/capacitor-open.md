# ionic capacitor open

打开指定原生平台项目的集成开发环境（IDE）

```shell
ionic capacitor open [options]
```

`ionic capacitor open` 将执行以下操作：

- 为你的原生项目打开对应的 IDE（iOS 使用 Xcode，Android 使用 Android Studio）
