# ionic integrations list

列出您应用中可用的和已激活的集成功能

```shell
ionic integrations list [options]
```

此命令将显示 Ionic 项目中各集成功能的状态。集成功能可能处于以下状态：**已启用**（已添加并启用）、**已禁用**（已添加但已禁用）和**未添加**（从未添加到项目中）。

- 若要启用或添加集成功能，请参阅 `ionic integrations enable --help`
- 若要禁用集成功能，请参阅 `ionic integrations disable --help`
