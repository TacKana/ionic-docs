# ionic completion

启用 Ionic CLI 命令的 Tab 自动补全功能。

```shell
ionic completion [options]
```

此命令为实验性功能，仅支持 Z shell (zsh) 和非 Windows 平台。

要为 Ionic CLI 启用自动补全，您可以将此命令打印的补全代码添加到您的 **~/.zshrc**（或任何其他由您的 shell 加载的文件）中。请参阅示例。

## 示例

```shell
$ ionic completion
$ ionic completion >> ~/.zshrc
```
