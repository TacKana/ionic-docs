# Ionic React 测试

当使用 Ionic CLI 生成 `@ionic/react` 应用时，系统会自动配置好使用 [Jest](https://jestjs.io/) 和 [React Testing Library](https://testing-library.com/docs/react-testing-library/intro) 进行单元测试和集成测试的环境。这套组合工具为单独测试 React 组件以及集成测试提供了绝佳的方式。有关如何开始测试 Ionic React 应用的更多信息，请参阅我们的文章 [使用 Jest 和 React Testing Library 测试 Ionic React 应用](https://ionicframework.com/blog/testing-ionic-react-apps-with-jest-and-react-testing-library)。
