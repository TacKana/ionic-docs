# ion-router-link

路由链接组件用于导航至指定链接。类似于浏览器的锚点标签，它可以接受用于定位的 href 属性和用于过渡动画的方向设置。

> 注意：此组件仅适用于原生 JavaScript 和 Stencil JavaScript 项目。对于 Angular 项目，请使用带有 Angular 路由器的 `<a>` 标签和 `routerLink` 指令。
