# ion-app

App 是 Ionic 应用程序的容器元素。每个项目中应该只有一个 `<ion-app>` 元素。一个应用可以包含许多 Ionic 组件，包括菜单、页眉、内容区域和页脚。当叠加层组件被展示时，它们会被附加到 `<ion-app>` 元素上。
