# ion-slide

Slide 组件是 [Slides](slides.md) 的子组件。模板中应使用 `ion-slide` 来编写。所有幻灯片内容都应写在此组件内，并且需要与 [Slides](slides.md) 配合使用。

更多使用信息，请参阅 [Slides API 文档](slides.md)。
