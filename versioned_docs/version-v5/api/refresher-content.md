# ion-refresher-content

`ion-refresher-content` 组件包含在下拉刷新时显示的文本、图标和旋转器。Ionic 会根据平台提供默认的下拉图标和刷新旋转器。然而，开发者可以根据刷新器的不同状态自定义默认的图标、旋转器和文本。
