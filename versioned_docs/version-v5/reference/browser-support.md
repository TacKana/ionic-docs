# 浏览器支持

Ionic 的最初目标是让使用 HTML、CSS 和 JavaScript 等 Web 技术开发移动应用变得简单。正是基于这一 Web 技术基础，Ionic 能够在 Web 运行的任何地方运行——iOS、Android、浏览器、PWA 等。

## 移动端浏览器

为了实现[自适应样式](../core-concepts/fundamentals.md#adaptive-styling)，Ionic 全面支持并已在以下移动平台进行了充分测试：

| 框架版本 | Android | iOS     |
| :-------: | :-----: | :-----: |
| Ionic V5  | 5.0+    | 11.0+   |
| Ionic V4  | 4.4+    | 10.0+   |

:::note
查看[最新 Android 统计数据](https://developer.android.com/about/dashboards/) 和[最新 iOS 统计数据](https://developer.apple.com/support/app-store/) 获取最新的平台信息。
:::

## 桌面端浏览器

由于 Ionic 基于 Web 技术，它在桌面浏览器上的表现与在移动设备上同样出色。有关桌面布局的更多信息，请参阅[跨平台](../core-concepts/cross-platform.md#desktop)。

| 浏览器     | Ionic v5 | Ionic V4 |
| :---------: | :------: | :------: |
| **Chrome**  | ✔        | ✔        |
| **Safari**  | ✔        | ✔        |
| **Edge**    | 79+      | ✔        |
| **Firefox** | ✔        | ✔        |
| **IE 11**   | **X**    | **X**    |
