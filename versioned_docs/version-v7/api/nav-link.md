导航链接用于跳转到指定的组件。可以通过`前进`、`后退`或设置为`根`组件的方式进行导航。

它是调用导航控制器上 `push()`、`pop()` 和 `setRoot()` 方法的元素形式。

更多信息请参阅 [nav](./nav#使用-navlink) 文档。
