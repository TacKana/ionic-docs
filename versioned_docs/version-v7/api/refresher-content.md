刷新器内容包含在下拉刷新期间显示的文本、图标和旋转器。Ionic 会根据平台显示拉动图标和刷新旋转器。不过，默认的图标、旋转器和文本可以根据刷新器的状态进行自定义。

有关使用示例，请参阅 [Refresher](/v7/api/refresher) 文档。
