浮动操作按钮列表组件是用于容纳多个[浮动操作按钮](./fab-button)的容器。它包含与主浮动操作按钮相关的操作，点击时会展开显示。要指定按钮出现在哪一侧，请将 `side` 属性设置为 `"start"`、`"end"`、`"top"` 或 `"bottom"`。

关于使用示例，请参阅[浮动操作按钮文档](./fab)。
