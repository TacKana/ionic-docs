# 从 Ionic 4 升级至 5

将应用程序从 Ionic 4 迁移到 5 需要对 API 属性、CSS 工具以及已安装的包依赖进行一些更新。

:::note
本指南假设您已将应用程序更新到 Ionic 4 的最新版本。在开始本指南之前，请确保您已遵循[升级至 Ionic 4 指南](./4-0)。
:::

:::info 重大变更
有关从 Ionic 4 到 Ionic 5 的**完整重大变更列表**，请参考 Ionic Framework 代码仓库中的[重大变更文档](https://github.com/ionic-team/ionic-framework/blob/main/BREAKING_ARCHIVE/v5.md)。
:::

### 包和依赖项

对于基于 Angular 的项目，您只需运行：

```shell
npm install @ionic/angular@v5-lts @ionic/angular-toolkit@4.0.0 --save
```

对于 React 项目，您可以运行：

```shell
npm install @ionic/react@v5-lts @ionic/react-router@v5-lts ionicons@5.5.3
```

对于 Stencil / 原生 JavaScript 项目，您可以运行：

```shell
npm i @ionic/core@v5-lts --save
```

如果您想要一个全新的项目起点，可以从 CLI 创建一个新的项目基础，并手动迁移现有的应用程序。
