### [使用 Ionic 和 Electron 构建桌面应用](https://pluralsight.pxf.io/VeMXO)

<!-- cspell:disable-next-line -->

作者：Michael Callaghan @ Pluralsight

历史上，桌面开发所需的技能与 Web 开发所需技能大相径庭，两者难以融合。在本课程《使用 Ionic 和 Electron 构建桌面应用》中，你将能够运用你辛苦获得的 Web 开发技能来构建出色的桌面应用程序。首先，你将学习如何使用 Ionic 和 Angular 构建功能齐全且美观的用户界面。接着，你将了解如何将该用户界面封装到 Electron 应用程序外壳中。最后，你将探索如何打包你的应用并使其准备好分发。完成本课程后，你将掌握 Ionic 和 Electron 开发所需的技能和知识，能够将精美的应用部署和分发给 Windows 和 macOS 用户。

### [使用 Ionic 构建渐进式 Web 应用](https://pluralsight.pxf.io/Ly2EY)

<!-- cspell:disable-next-line -->

作者：Michael Callaghan @ Pluralsight

当谷歌提出渐进式 Web 应用（PWA）的概念时，一切都改变了。PWA 是一种纯粹的 Web 应用程序，你可以将其安装在设备上，并通过智能缓存的使用，在有限的网络功能下运行。构建一个能在任何地方运行的渐进式 Web 应用。在本课程《使用 Ionic 构建渐进式 Web 应用》中，你将学习基础知识，并掌握创建能够在任何地方（浏览器、桌面或移动客户端）运行的 Web 应用程序的能力。首先，你将了解什么是渐进式 Web 应用。接下来，你将学习如何使用 Ionic 框架、Angular 和 Firebase，将一个基本的 Web 应用程序创建、部署和优化为一个成熟的 PWA。最后，你将探索如何配置应用程序，使其能够在 Android 和 iPhone 上安装和运行。完成本课程后，你将掌握创建和部署自己的渐进式 Web 应用到任何所需位置所需的 Ionic 和 PWA 技能与知识。

### [Ionic CLI](https://pluralsight.pxf.io/ionic-cli)

<!-- cspell:disable-next-line -->

作者：Michael Callaghan @ Pluralsight

自其诞生以来，Ionic 框架就包含了一个基础的命令行界面。虽然只有短短几年，但它已经发展成为一个强大的工具，应该成为每个开发者工具箱的一部分。本课程《Ionic CLI》将从顶层开始探索 Ionic CLI。首先，你将学习如何创建项目和组件。接下来，你将学习如何构建和提供应用服务。最后，你将发现如何与他人共享项目，甚至与其他构建工具集成。无论你是刚刚开始探索 Ionic，还是从它的预测试版时代就开始使用它，这里都有适合你的内容。到课程结束时，你将充满信心地将 Ionic CLI 作为日常 Ionic 开发的一部分。
