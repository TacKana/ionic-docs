# 书籍推荐

### [《从零开始学习 Ionic 4》](https://leanpub.com/learnionic4fromscratch)

Angular、Vue、React、原生 JavaScript。得益于全新的 Stencil 编译器，这些工具现在都可以用来创建出色的 Ionic 应用程序。本书面向初学者，旨在通过流行的框架示例，指导大家使用 Ionic 创建卓越的 Web、移动和桌面应用程序。

<!-- cspell:disable-next-line -->

作者：[Paul Halliday](https://developer.school)

### [《使用 StencilJS 创建 Ionic 应用程序》](https://www.joshmorony.com/creating-ionic-applications-with-stencil-js/) - [免费预览](https://cdn2.hubspot.net/hubfs/3776657/PREVIEW-Creating-Ionic-Apps-with-StencilJS.pdf)

StencilJS 不仅是生成可复用 Web 组件的强大工具，还提供了构建完整 Web 组件应用程序所需的一切。结合 Ionic 的 Web 组件，StencilJS 为我们提供了构建高质量生产级移动应用程序所需的所有功能——无需任何框架。

<!-- cspell:disable-next-line -->

作者：[Joshua Morony](https://www.joshmorony.com/blog)

### [《使用 Ionic 进行移动应用开发：基于 Ionic 2、Angular 2 和 Cordova 的跨平台应用》](https://www.amazon.com/Mobile-App-Development-Ionic-Cross-Platform/dp/1491937785/ref=sr_1_2?ie=UTF8&qid=1464183332&sr=8-2&keywords=ionic+2)

<!-- cspell:disable-next-line -->

### [《使用 Ionic 和 Angular 构建移动应用》](https://www.joshmorony.com/building-mobile-apps-with-ionic-2/)

<!-- cspell:disable-next-line -->

作者：[Joshua Morony](https://www.joshmorony.com/blog)

《使用 Ionic 和 Angular 构建移动应用》是一本学习最新版本 Ionic 的综合性资源。它面向初学者，从 Ionic 基础知识开始，逐步讲解不同复杂度的示例应用，最后指导如何构建和发布应用程序（到应用商店或作为 PWA）。本书会随着每个主要版本进行更新，因此您可以放心学习，无需担心技术过时。
