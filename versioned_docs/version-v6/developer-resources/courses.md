### [使用 Ionic 和 Electron 构建桌面应用](https://pluralsight.pxf.io/VeMXO)

<!-- cspell:disable-next-line -->

作者：Michael Callaghan（于 Pluralsight）

桌面开发历来需要与 Web 开发截然不同的技能，两者难以融合。在这门《使用 Ionic 和 Electron 构建桌面应用》课程中，你将能够运用辛苦获得的 Web 开发技能来构建出色的桌面应用程序。首先，你将学习如何使用 Ionic 和 Angular 构建功能完善且美观的用户界面。接着，你将了解如何将该界面封装到 Electron 应用外壳中。最后，你将探索如何打包应用程序并使其准备好分发。完成本课程后，你将具备 Ionic 和 Electron 开发的技能和知识，能够向 Windows 和 macOS 用户部署和分发美观的应用程序。

### [使用 Ionic 构建渐进式 Web 应用](https://pluralsight.pxf.io/Ly2EY)

<!-- cspell:disable-next-line -->

作者：Michael Callaghan（于 Pluralsight）

当谷歌提出渐进式 Web 应用（PWA）的概念后，一切发生了变化。PWA 是一种纯粹的 Web 应用程序，可以安装在设备上，并通过智能缓存实现在有限网络功能下的运行。构建一个可在任何地方运行的渐进式 Web 应用。在本课程《使用 Ionic 构建渐进式 Web 应用》中，你将学习基础知识并获得创建可在任何地方（浏览器、桌面或移动客户端）运行的 Web 应用程序的能力。首先，你将了解什么是渐进式 Web 应用（PWA）。接着，你将发现如何使用 Ionic 框架、Angular 和 Firebase 将一个基础的 Web 应用创建、部署和优化为成熟的 PWA。最后，你将探索如何配置应用程序，使其可在 Android 和 iPhone 上安装和运行。完成本课程后，你将具备创建和部署自己的渐进式 Web 应用所需的 Ionic 和 PWA 技能与知识，随心所欲地将其部署到任何地方。

### [Ionic CLI](https://pluralsight.pxf.io/ionic-cli)

<!-- cspell:disable-next-line -->

作者：Michael Callaghan（于 Pluralsight）

自诞生以来，Ionic 框架就包含了一个基础的命令行界面。虽然仅诞生数年，但它已发展成为一个强大的工具，应成为每位开发者工具箱的一部分。本课程《Ionic CLI》将从顶层开始探索 Ionic CLI。首先，你将了解如何创建项目和组件。接着，你将学习如何构建和运行应用程序。最后，你将发现如何与他人共享项目，甚至与其他构建工具集成。无论你是刚开始探索 Ionic，还是从测试版之前就开始使用它，这里都有适合你的内容。课程结束时，你将充满信心地将 Ionic CLI 作为日常 Ionic 开发的一部分。
