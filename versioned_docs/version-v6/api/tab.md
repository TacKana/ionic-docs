选项卡组件是 [tabs](tabs.md) 的子组件。每个选项卡可以包含一个应用顶级的导航栈或单个视图。一个应用可以拥有多个选项卡，每个都有自己独立的导航。

:::note
注意：该组件仅适用于原生 JavaScript 项目。对于 Angular、React 和 Vue 应用，您不需要使用 `ion-tab` 来声明您的选项卡组件。
:::

有关配置选项卡的更多详细信息，请参阅 [tabs 文档](tabs.md)。
