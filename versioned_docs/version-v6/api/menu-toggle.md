Menu Toggle 组件可用于切换菜单的打开或关闭状态。

菜单切换按钮仅在所选菜单启用时可见。如果菜单被禁用，或者它被呈现为分割面板，菜单切换按钮将被隐藏。如需始终显示菜单切换按钮，可将 `autoHide` 属性设为 `false`。

更多信息请参阅 [menu](./menu#menu-toggle) 文档。
