Icon 是一个通过 <a href="https://ionic.io/ionicons">Ionicons</a> 库提供的简单组件，所有 Ionic Framework 应用默认都会预装此组件。它可以用来显示 Ionicons 集合中的任何图标，或者自定义的 SVG 图标。同时支持尺寸和颜色等样式设置。

要查看所有可用图标，请访问 <a href="https://ionic.io/ionicons">ionic.io/ionicons</a>。如需了解更多信息，包括样式设置和自定义 SVG 使用方法，请参阅 <a href="https://ionic.io/ionicons/usage">使用说明页面</a>。
