# 测试 Ionic React 应用

当使用 Ionic CLI 生成 `@ionic/react` 应用时，系统会自动配置好基于 [Jest](https://jestjs.io/) 和 [React Testing Library](https://testing-library.com/docs/react-testing-library/intro) 的单元测试和集成测试环境。这套组合能出色地支持对 React 组件进行独立测试和集成测试。有关如何开始测试 Ionic React 应用的更多信息，请参阅我们的文章 [使用 Jest 和 React Testing Library 测试 Ionic React 应用](https://ionicframework.com/blog/testing-ionic-react-apps-with-jest-and-react-testing-library/)。
